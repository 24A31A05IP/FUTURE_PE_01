Prompt Logic

The prompt takes the business name, business type, and location as inputs.
Based on these details, it generates professional website content tailored to the selected local business. 
The content includes a homepage section, service descriptions, and a persuasive call-to-action section. 
The prompt is designed to adapt its tone and messaging according to the business category.

Generated Outputs

1. Homepage Copy

- Headline
- Subheadline
- Introduction

2. Services Page Content

- Service Name
- Service Description

3. CTA Section

- Call-to-Action Heading
- Persuasive CTA Content

Conclusion

The generated website content is ready to be used on a real business website.
The framework can be easily adapted for different local businesses such as cafes, salons, clinics, coaching institutes
and small agencies by simply changing the business details provided in the prompt.
